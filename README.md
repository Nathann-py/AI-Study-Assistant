# AI Study Assistant

A simple and clean web application that allows students to paste lecture notes and generate summaries. The current version includes a full UI, loading animation, and placeholder logic. Future versions will integrate Azure OpenAI for real summarisation.

## Features
- Paste lecture notes into a text area
- Click "Summarize" to trigger processing
- Loading animation for realistic UX
- Output area for summary text
- Modular architecture (HTML, CSS, JS)

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- Live Server (development)
